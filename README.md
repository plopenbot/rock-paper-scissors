# 石头剪刀布游戏 🎮

一个使用 HTML、CSS 和 JavaScript 开发的经典石头剪刀布游戏。

## 🎯 游戏特点

- **精美界面**：现代化的卡片式设计，渐变背景
- **动画效果**：流畅的按钮交互和结果展示动画
- **实时计分**：自动记录玩家胜利、平局和敌人胜利次数
- **响应式设计**：支持各种屏幕尺寸

## 🚀 在线体验

访问: [在线游戏地址](https://pispeng.github.io/rock-paper-scissors/)

## 📦 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/pispeng/rock-paper-scissors.git
cd rock-paper-scissors
```

2. 直接在浏览器中打开 `index.html` 文件即可开始游戏

## 🎮 游戏规则

- 石头 🪨 克制 剪刀 ✂️
- 剪刀 ✂️ 克制 布 📄
- 布 📄 克制 石头 🪨

## 🛠️ 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 渐变, 动画)
- Vanilla JavaScript
- 响应式设计

## 📊 游戏记录

游戏会自动记录：
- 玩家胜利次数
- 平局次数
- 敌人胜利次数
