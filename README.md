# 石头剪刀布游戏 🎮

一个使用 HTML、CSS 和 JavaScript 开发的经典石头剪刀布游戏，完美适配 iPhone 15 Pro Max。

## 🎯 游戏特点

- **精美界面**：现代化的卡片式设计，渐变背景
- **动画效果**：流畅的按钮交互和结果展示动画
- **实时计分**：自动记录玩家胜利、平局和敌人胜利次数
- **📱 移动端优化**：专为 iPhone 15 Pro Max 优化，完美支持触摸操作
- **响应式设计**：支持各种屏幕尺寸和横竖屏切换

## 📱 移动端特性

### iPhone 优化
- ✅ 全屏沉浸式体验（隐藏地址栏）
- ✅ 禁用双击缩放，防止误触
- ✅ 优化触摸反馈（无高亮闪烁）
- ✅ 按钮大小适合手指点击（无需精确点击）
- ✅ iOS 原生字体渲染
- ✅ 状态栏透明样式

### 自适应布局
- 📐 竖屏：舒适的三列按钮布局
- 🔄 横屏：紧凑的横向显示
- 📏 小屏幕：自动缩小元素尺寸
- 📏 大屏幕：放大显示更清晰

## 🚀 在线体验

**移动端访问（推荐）：**

用你的 iPhone 扫描二维码或直接访问链接，添加到主屏幕获得最佳体验！

## 📦 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/plopenbot/rock-paper-scissors.git
cd rock-paper-scissors
```

2. 直接在浏览器中打开 `index.html` 文件即可开始游戏

## 🎮 游戏规则

- 石头 🪨 克制 剪刀 ✂️
- 剪刀 ✂️ 克制 布 📄
- 布 📄 克制 石头 🪨

## 🛠️ 技术栈

- HTML5
- CSS3 (Grid, Flexbox, 渐变, 动画, 媒体查询)
- Vanilla JavaScript
- 响应式设计
- 移动端触摸优化

## 📊 游戏记录

游戏会自动记录：
- 玩家胜利次数
- 平局次数
- 敌人胜利次数
- 完整的对局历史（带滚动）

## 🎨 设计亮点

- **渐变背景**：紫色渐变背景提供视觉吸引力
- **卡片设计**：白色圆角卡片，带阴影效果
- **彩色图标**：蓝色石头、绿色剪刀、黄色布
- **动画反馈**：按钮点击缩放、失败震动效果
- **结果标识**：胜利（绿色）、失败（红色）、平局（橙色）

## 📱 移动端最佳实践

1. **添加到主屏幕**：在 Safari 中点击"分享" → "添加到主屏幕"
2. **全屏模式**：从主屏幕启动，获得无地址栏的沉浸式体验
3. **横竖屏**：支持自动旋转，横屏紧凑模式方便单手操作
4. **流畅滚动**：历史记录支持 iOS 原生滚动体验
